# goit-js-hw-03

### Про модуль

## Цей модуль допомагає закріпити базові навички роботи з масивами та функціями в JavaScript:

    •	Розуміння принципів роботи масивів.
    •	Використання базових методів масивів.
    •	Особливості застосування циклу for…of для ітерації по масиву.
    •	Можливості роботи з функціями.
    •	Розуміння різниці між глобальною та блочною областями видимості.
